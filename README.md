# Django Soccer Video Manager

Soccer video management web application built with Python and Django.  
The project allows users to manage soccer-related video records using Django views, templates, static files and a structured backend project setup.

## Features

- Manage soccer video records
- Render pages using Django templates
- Organize static files for frontend assets
- Use Django project structure with settings and URL routing
- Separate backend configuration from templates and static content
- Basic web application flow
- Project created for practicing Django backend development

## Installation

Clone the repository:

```bash
git clone https://github.com/ArathJacobo061/django-soccer-video-manager.git
```

Go to the project folder:

```bash
cd django-soccer-video-manager
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment on Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start the development server:

```bash
python manage.py runserver
```

Open the app in your browser:

```txt
http://127.0.0.1:8000/
```
